# Atlas

## Deconstructing the AI Agent: My Journey with Atlas

I wanted to understand how tools like Cursor and Claude Code actually work under the hoodâ€”so I decided to build one myself.

Atlas is a lightweight, agentic code editor built in Python. Instead of using pre-made frameworks that hide the complexity, I built this agent from the ground up using the Google Gemini API.

The project explores the critical transition from "Chatbot" to "Agent":

  1. Giving the model tools (Function Calling).

  2. Allowing it to execute code.

  3. Creating a Feedback Loop where the agent can catch its own bugs and fix them in real-time.

Current State: The project currently explores the critical transition from "Chatbot" to "Agent" via function calling and error-correction loops.
